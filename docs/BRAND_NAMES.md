# 应用命名方案

## 中文主推荐

### 妙影工坊

推荐作为当前应用正式中文名。

理由：

- 「妙影」有 AI 影像、艺术写真、生成惊喜的联想。
- 「工坊」比「工具」更有创作感，也适合后续扩展到头像、海报、写真套系。
- 读起来顺口，适合微信小程序、公众号、分享海报和管理后台展示。

推荐副标题：

```text
上传一张照片，生成你的 AI 艺术写真
```

| 一瞬成片 | 强调快速生成，适合传播和转化 |
| 造梦影像馆 | 情绪价值更强，适合亲子、头像、朋友圈写真 |
| 闪耀写真局 | 年轻轻快，适合社交分享 |
| 千面影像馆 | 强调多风格、多版本的自己 |
| 灵感写真馆 | 稳妥、文艺、高级感 |

| VividMe | `vivid-me` | 强调「更鲜活的自己」 |

## 工程命名建议

优先使用：

```text
magic-frame
```

相关资源可以统一命名：

```text
magic-frame-weapp
magic-frame-backend
magic-frame-admin
magic-frame-cloudrun
magic-frame-assets
```

如果希望中文品牌和工程名关联更强，可以使用：

```text
miao-frame
```

不建议工程名使用中文拼音全称 `miaoyinggongfang`，长度偏长，不利于命令行、服务名和对象存储路径。
